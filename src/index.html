Traffic Report (Teachers' Edition)

Rails API, React/Redux

An extension for a school's online learning platform, Traffic Report not only serves as a place for students to check their nightly homework, keep up with class announcements, and ask questions in real-time; it also prompts students to self-report their current comfort with class material. Traffic Report gathers data about these fluctuating levels of understanding and provides it to teachers. Graphical representations of the data allow teachers to visualize trends, track a student's progress, and adjust lessons accordingly.

Stellar: An App for the Stars

Rails API, JavaScript, React.js

Stellar provides a simple interface to explore upcoming astronomical events and add ones of interest to your personal calendar. Browse the constellations to learn more about them; check the moon phase over the next few weeks; or search for future meteor showers and fireballs. Simply add an event from our collection of these phenomena (provided by NASA and others), jot down any notes you have for yourself, and get ready to enjoy the stars.
